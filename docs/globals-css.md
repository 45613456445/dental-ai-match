# globals.css - カラーシステム

## 概要
DentalAI Match のグローバルスタイル定義ファイル。
Tailwind CSS v4 + CSS変数でカラーテーマを管理している。

## 対応ファイル
`src/app/globals.css`

## カラーパレット

### Primary - Warm Rose（ローズ系）
メインカラー。ボタン、リンク、ヒーローセクション、アクセントに使用。

| 変数名 | Tailwind クラス | 用途 |
|--------|----------------|------|
| `--color-primary-50` (#fff1f2) | `rose-50` | 薄い背景 |
| `--color-primary-100` (#ffe4e6) | `rose-100` | ホバー背景 |
| `--color-primary-200` (#fecdd3) | `rose-200` | ボーダー |
| `--color-primary-500` (#f43f5e) | `rose-500` | フォーカスリング |
| `--color-primary-600` (#e11d48) | `rose-600` | ボタン、リンク |
| `--color-primary-700` (#be123c) | `rose-700` | ヒーロー背景、ホバー |

### Accent - Teal（ティール系）
サブカラー。医院向けCTAボタン、歯科医師バッジ、ステップ表示に使用。

| 変数名 | Tailwind クラス | 用途 |
|--------|----------------|------|
| `--color-teal-50` (#f0fdfa) | `teal-50` | 薄い背景 |
| `--color-teal-300` (#5eead4) | `teal-300` | ヒーロー上のアクセントテキスト |
| `--color-teal-400` (#2dd4bf) | `teal-400` | CTAボタン |
| `--color-teal-500` (#14b8a6) | `teal-500` | ステップ番号 |
| `--color-teal-700` (#0f766e) | `teal-700` | テキスト |

### その他
- **背景**: `#fdf8f6`（温かみのあるオフホワイト）
- **文字色**: `#1a1a2e`（ダークネイビー）
- **セレクション**: `#ffe4e6`（ローズ系ハイライト）
- **フォーカス**: `#f43f5e`（ローズ500）

## デザイン方針
- 「温かい・親しみやすい」印象を目指す配色
- 高齢者にも読みやすいフォントサイズ（16px基本）を維持
- 行間1.8、見出し行間1.4で可読性を確保
